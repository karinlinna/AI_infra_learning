## 3. 方向定位：主攻 + 辅攻（细化到岗位 title）

### 🎯 主攻：LLM 应用开发 / AI Agent 开发工程师
- **对应 title**：AI 应用开发工程师、大模型应用工程师、Agent 开发工程师、RAG 工程师、AIGC 应用开发。
- **为什么选它**：
  - 你的 Tesla RAG + Agent 研究 = **直接对口**，简历不用大改就能投；
  - 岗位量大（几乎所有大厂 + AI 创业公司都在招），应届友好；
  - 补齐成本最低——补的是"工程落地 + 一个能讲的 Agent 项目"，不是从零学后端或分布式训练。

### 🎯 辅攻一：算法工程师（NLP / LLM 应用算法）
- **对应 title**：NLP 算法工程师、大模型算法工程师（应用向）、搜索/推荐算法（含 RAG 检索）。
- **为什么**：硕士 + ML 背景够门槛，但需**补回算法基础 + 刷题**，作为"冲高"选项。

### 🎯 辅攻二：AI 产品经理 / AI 解决方案（兜底 + 差异化）
- **对应 title**：AI 产品经理、大模型解决方案工程师、AI Forward Deployed Engineer。
- **为什么**：10 个月产品实习 + PRD 能力是现成弹药，技术岗全线不顺时的**高质量兜底**。

### ❌ 明确不主投
- **纯后端开发**（元宝后台这类）：ML 背景不占优，要补的后端栈太重，性价比低。
- **Agent 后训练算法**：要求"1 年后训练经验 + RLHF/RM/分布式训练"，是**社招门槛**，应届极难直接进。（可作为"进阶目标"，不是第一份工作目标。）
- **CV 算法**：和市场热点错位，你的项目也不在 CV。

> **你的方向拍板（要自己认同，后续才有动力）：**
> ```
> 主攻：LLM 应用 / AI Agent 开发工程师
> 辅攻①：NLP / 大模型应用算法工程师
> 辅攻②：AI 产品 / 解决方案（兜底）
> 地域：北上广深杭 + 青岛等优质新一线
> 目标薪资：月薪 2–3w（认可技术岗前期低、后期涨幅大）
> ```

## 5. 提升路线：4–5 节课怎么补

> **总策略**：把 RAG 长板做成能讲透的作品，再补一个完整的 Agent 应用项目 + 部署上线，同时并行捡回算法基础和刷题。三个月内让简历从"有基础"变成"**有作品、能上线、能讲清楚**"。

| 课 | 主题 | 你上完要达到 |
|---|---|---|
| **第 2 节** | Agent 工程实战（`AIagent` 9 课）| 不看教程独立写出带工具+记忆的 Agent；说清 Agent Loop 每一步 |
| **第 3 节** | 落地可上线项目（RAG+Agent+FastAPI+Docker）| 有一个能跑、含架构图 README 的 GitHub 项目；补上后端部署 |
| **第 4 节** | 算法基础回补 + 面试八股 | 白板讲清 Attention & LoRA；备好 20 道八股标准答案 |
| **第 5 节** | 简历精修 + 模拟面试 + 投递 SOP | 可直投简历 + 2 分钟项目高光 + 投递清单 |
| （可选）第 6 节 | 专项冲刺 | 算法刷题 / 微调后训练进阶（二选一）|
