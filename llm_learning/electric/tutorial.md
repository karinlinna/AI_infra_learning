## 1. 这个项目是做什么的？

**一句话概括**：用牛顿-拉夫逊（Newton-Raphson）方法，计算电力系统中每个节点的电压，以及每条线路上的功率流向。

### 现实中的对应

想象你家小区的供电网络：

- 发电厂发出电 -> 通过高压线路传输 -> 经过变压器降压 -> 送到你家
- 每条线路有电阻，会导致**电压降低**和**功率损耗**
- 电力公司需要知道：每个节点的电压是多少？每条线路传输了多少功率？有没有超过安全范围？

**潮流计算（Power Flow Analysis）** 就是回答这些问题的核心工具。

### 为什么不能直接用欧姆定律算？

因为电力系统是**非线性**的。功率公式 `P = V * I * cos(θ)` 中，电压 V 本身是未知数，而且所有节点的电压互相影响。所以需要用**迭代法**（一步步逼近正确答案）来求解。

## 3. 前置知识：电力系统基础概念

### 3.1 标幺值（Per-Unit, pu）

电力系统中，不同位置的电压等级差异很大（110kV、220kV、500kV...），直接用实际值计算很麻烦。

**标幺值**就是把所有数值除以一个"基准值"，变成无量纲的比例数：

```
标幺值 = 实际值 / 基准值
```

**举例**：
- 基准电压 = 220kV
- 某节点实际电压 = 231kV
- 标幺值 = 231 / 220 = **1.05 pu**

**好处**：
- 不同电压等级的数据可以直接比较
- 正常电压都在 **0.95 ~ 1.05 pu** 范围内，一眼就能看出是否正常
- 本项目中所有数据都用标幺值

### 3.2 有功功率（P）和无功功率（Q）

- **有功功率 P**：真正做功的功率（点亮灯泡、驱动电机），单位 MW
- **无功功率 Q**：不做功，但维持电磁场所必需的功率，单位 Mvar
- **复功率 S = P + jQ**：用复数同时表示两者

```
类比：
- 有功功率 P = 啤酒中你能喝到的液体部分
- 无功功率 Q = 啤酒上面的泡沫
- 视在功率 |S| = 整杯啤酒的体积
```

### 3.3 三种节点类型

电力系统中的每个节点（母线/Bus）分为三种类型：

| 类型 | 英文 | 已知量 | 未知量 | 说明 |
|------|------|--------|--------|------|
| **PQ 节点** | Load Bus | P, Q | V, θ | 负荷节点，用电量已知 |
| **PV 节点** | Generator Bus | P, V | Q, θ | 发电机节点，发电量和电压已知 |
| **平衡节点** | Slack Bus | V, θ | P, Q | 参考节点，补偿系统功率不平衡 |

**为什么需要平衡节点？**

在计算之前，我们不知道线路上的损耗是多少，所以所有发电机的发电量之和不一定等于所有负荷的用电量之和。平衡节点就是那个"多退少补"的角色——它的发电量会自动调整，确保整个系统功率平衡。

### 3.4 导纳矩阵（Y-Bus Matrix）

**导纳 Y = 1/阻抗 = 1/(R + jX)**，其中：
- R = 电阻（消耗功率）
- X = 电抗（储存/释放能量）
- j = 虚数单位（表示相位关系）

**导纳矩阵**是描述整个电网连接关系的核心矩阵：

```
对于一个3节点系统，Y矩阵是 3x3 的：

Y = | Y11  Y12  Y13 |
    | Y21  Y22  Y23 |
    | Y31  Y32  Y33 |

其中：
- Y12 = -y12（节点1和2之间线路的导纳，取负号）
- Y11 = y12 + y13 + ...（与节点1相连的所有线路导纳之和）
```

**规则**：
- **对角线元素** Y_ii = 与节点 i 相连的所有支路导纳之和
- **非对角线元素** Y_ij = 节点 i 和 j 之间支路导纳的**负值**
- 矩阵是**对称的**：Y_ij = Y_ji

### 3.5 潮流方程

每个节点的功率注入可以用电压和导纳矩阵表示：

```
P_i = Σ |V_i| * |V_j| * (G_ij * cos(θ_i - θ_j) + B_ij * sin(θ_i - θ_j))
Q_i = Σ |V_i| * |V_j| * (G_ij * sin(θ_i - θ_j) - B_ij * cos(θ_i - θ_j))
```

其中：
- |V_i|, θ_i = 节点 i 的电压幅值和相角
- G_ij + jB_ij = Y_ij（导纳矩阵的元素，G是电导，B是电纳）
- Σ 表示对所有节点 j 求和

**这就是需要求解的非线性方程组**——方程中的未知数（V 和 θ）既在三角函数里，又在乘积中，无法直接求解。

### 3.6 牛顿-拉夫逊法（Newton-Raphson Method）

牛顿-拉夫逊法是一种用于**求解非线性方程根**的迭代数值方法，即求 `f(x) = 0` 的解。

#### 通用数学原理

**迭代公式**：

```
x_{n+1} = x_n - f(x_n) / f'(x_n)
```

其中 `f'(x_n)` 是函数在 `x_n` 处的导数（即切线斜率）。

**直观理解**：

1. 在当前点 `x_n` 画出函数的切线
2. 切线与 x 轴的交点就是下一个近似值 `x_{n+1}`
3. 重复此过程，逐步逼近真正的根

**简单示例：求 √2（即求 x² - 2 = 0 的正根）**

- `f(x) = x² - 2`，`f'(x) = 2x`
- 迭代公式：`x_{n+1} = x_n - (x_n² - 2) / (2x_n)`
- 取 `x₀ = 1`：
  - `x₁ = 1 - (1-2)/(2) = 1.5`
  - `x₂ = 1.5 - (2.25-2)/(3) = 1.4167`
  - `x₃ = 1.4142` ← 已经非常接近 √2

**优缺点**：

| 优点 | 缺点 |
|------|------|
| 收敛速度快（二次收敛） | 需要计算导数 `f'(x)` |
| 实现简单 | 初始值选不好可能不收敛 |
| 适用范围广 | `f'(x) = 0` 时公式失效 |

#### 在潮流计算中的应用

**核心思想**：从一个初始猜测值开始，通过不断修正，逐步逼近真实解。

在电力系统潮流计算中，`f(x)` 对应的是多维的功率失配方程，`f'(x)` 对应的是**雅可比矩阵（Jacobian）**，`x` 则是所有待求的电压幅值和相角组成的向量。单变量公式推广为：

```
Δx = J⁻¹ · mismatch    （即解线性方程组 J · Δx = mismatch）
x_{n+1} = x_n + Δx
```

**具体步骤**：

```
1. 猜一个初始值（通常 V=1.0, θ=0°）
2. 用当前猜测值代入方程，计算功率
3. 比较计算值和已知值的差距（称为"失配量"）
4. 如果差距足够小 → 结束，找到答案
5. 否则 → 计算雅可比矩阵（Jacobian，描述"每个变量变化一点点，功率会变多少"）
6. 解线性方程组，得到修正量
7. 更新猜测值，回到第2步
```

**雅可比矩阵**是什么？

```
J = | ∂P/∂θ    ∂P/∂V |    即"功率对电压的敏感度"
    | ∂Q/∂θ    ∂Q/∂V |

含义：
- ∂P/∂θ：角度变化1度，有功功率变化多少？
- ∂P/∂V：电压变化0.01pu，有功功率变化多少？
- ...以此类推
```

**牛顿法的优点**：**二次收敛**——每迭代一次，误差大约变为原来的平方。

```
迭代1: 误差 = 0.1
迭代2: 误差 ≈ 0.01      (0.1²)
迭代3: 误差 ≈ 0.0001    (0.01²)
迭代4: 误差 ≈ 0.00000001 (0.0001²)
```

通常 3~5 次迭代就能达到很高精度。

## 4. 模块一：数据结构 (network.py)

这个文件定义了电力系统的"地图"——节点是什么、线路是什么、它们怎么连接。

### 4.1 BusType 枚举

```python
class BusType(Enum):
    PQ = 1      # 负荷节点
    PV = 2      # 发电机节点
    SLACK = 3   # 平衡节点
```

用 Python 的枚举类型，让代码中的节点类型更清晰易读。

### 4.2 Bus 类（节点）

```python
class Bus:
    def __init__(self, bus_id, bus_type=BusType.PQ, ...):
        self.bus_id = bus_id          # 节点编号（从1开始）
        self.bus_type = bus_type      # 节点类型
        self.p_load = p_load          # 有功负荷（用电量）
        self.q_load = q_load          # 无功负荷
        self.p_gen = p_gen            # 有功发电
        self.q_gen = q_gen            # 无功发电
        self.v_mag = v_mag            # 电压幅值
        self.v_ang = v_ang            # 电压相角（弧度）
        self.q_min = q_min            # 无功下限
        self.q_max = q_max            # 无功上限
        self.b_shunt = b_shunt        # 并联电纳（电容器组）
```

**关键属性**：

```python
@property
def p_net(self):
    return self.p_gen - self.p_load   # 净有功功率 = 发电 - 用电

@property
def q_net(self):
    return self.q_gen - self.q_load   # 净无功功率 = 发电 - 用电
```

`p_net` > 0 表示该节点是净发电（向外输出功率），< 0 表示是净负荷（从外部吸收功率）。

### 4.3 Branch 类（支路/线路）

```python
class Branch:
    def __init__(self, from_bus, to_bus, r, x, b=0, tap=1.0):
        self.from_bus = from_bus   # 起始节点编号
        self.to_bus = to_bus       # 终止节点编号
        self.r = r                 # 电阻（pu）
        self.x = x                 # 电抗（pu）
        self.b = b                 # 线路充电电纳（pu）
        self.tap = tap             # 变压器变比（普通线路为1.0）
```

**关键属性**：

```python
@property
def z(self):
    return complex(self.r, self.x)   # 复数阻抗 Z = R + jX

@property
def y(self):
    return 1.0 / self.z              # 导纳 Y = 1/Z
```

**关于线路模型（π 型等值电路）**：

```
节点i  ──────[Z = R + jX]──────  节点j
   |                                |
  jb/2                            jb/2
   |                                |
  地                               地
```

- 中间的 Z 代表线路阻抗（电阻产生损耗，电抗产生压降）
- 两端的 jb/2 代表线路对地充电电容（长线路才明显）
- 变压器时 tap ≠ 1.0，会按变比修正导纳

### 4.4 PowerNetwork 类（电网）

```python
class PowerNetwork:
    def __init__(self):
        self.buses = {}       # 所有节点 {bus_id: Bus对象}
        self.branches = []    # 所有支路列表
        self.y_bus = None     # 导纳矩阵（待构建）
```

**最核心的方法 `build_y_bus()`**——构建导纳矩阵：

```python
def build_y_bus(self):
    n = self.n_bus
    Y = np.zeros((n, n), dtype=complex)  # 初始化全零矩阵

    for branch in self.branches:
        i = idx_map[branch.from_bus]  # 节点编号 -> 矩阵下标
        j = idx_map[branch.to_bus]
        y = branch.y                  # 支路导纳
        t = branch.tap                # 变压器变比

        # 对角线：加上与该节点相连的支路导纳
        Y[i, i] += y / (t * t) + 1j * branch.b / 2
        Y[j, j] += y + 1j * branch.b / 2

        # 非对角线：取负值
        Y[i, j] -= y / t
        Y[j, i] -= y / t

    self.y_bus = Y
```

**图解**（以3节点系统为例）：

```
      节点1 ----y12---- 节点2
        \                /
        y13            y23
          \            /
           节点3

Y矩阵 = | (y12+y13)    -y12        -y13      |
         | -y12         (y12+y23)   -y23      |
         | -y13         -y23        (y13+y23) |
```

## 5. 模块二：核心算法 (newton_raphson.py)

这是整个项目最核心的部分，实现了牛顿-拉夫逊潮流计算。

### 5.1 主函数 `newton_raphson_power_flow()`

**输入**：
- `network`：已构建好 Y 矩阵的电网对象
- `tol`：收敛容差（默认 1e-6，越小越精确）
- `max_iter`：最大迭代次数（默认 30）

**输出**（字典）：
- `converged`：是否收敛
- `v_mag`、`v_ang`：每个节点的电压幅值和相角
- `branch_flows`：每条线路的功率流
- `losses`：总损耗
- `mismatch_history`：每次迭代的最大失配量（用于画收敛曲线）

### 5.2 算法流程详解

#### 第一步：初始化

```python
# 电压幅值：PV和SLACK节点用给定值，PQ节点默认1.0
v_mag = np.array([bus.v_mag for bus in buses])

# 电压相角：所有节点初始为0（或给定值）
v_ang = np.array([bus.v_ang for bus in buses])

# 获取各类节点在矩阵中的下标
pq_idx = network.get_pq_indices()    # PQ节点的下标列表
pv_idx = network.get_pv_indices()    # PV节点的下标列表
pvpq_idx = np.concatenate([pv_idx, pq_idx])  # PV+PQ节点
```

#### 第二步：迭代循环

每次迭代做以下事情：

**2a. 计算功率注入**

```python
p_calc, q_calc = _calc_power_injection(v_mag, v_ang, Y)
```

用当前的电压估计值，代入潮流方程，算出每个节点"应该"注入多少功率。

**2b. 计算失配量**

```python
# 有功失配 = 给定值 - 计算值（对PV和PQ节点）
dp = p_spec[pvpq_idx] - p_calc[pvpq_idx]

# 无功失配 = 给定值 - 计算值（仅对PQ节点）
dq = q_spec[pq_idx] - q_calc[pq_idx]

# 组合成一个大向量
mismatch = np.concatenate([dp, dq])
max_mismatch = np.max(np.abs(mismatch))
```

**失配量 = "应该是多少" - "按当前电压算出来是多少"**

如果失配量足够小（< tol），说明当前电压就是答案。

**2c. 构建雅可比矩阵**

```python
J = _build_jacobian(v_mag, v_ang, Y, pvpq_idx, pq_idx)
```

雅可比矩阵是一个大矩阵，包含4个子矩阵：

```
J = | J1  J2 |
    | J3  J4 |

J1 = ∂P/∂θ（角度变一点，有功变多少）—— 大小: (n_pvpq × n_pvpq)
J2 = ∂P/∂V（电压变一点，有功变多少）—— 大小: (n_pvpq × n_pq)
J3 = ∂Q/∂θ（角度变一点，无功变多少）—— 大小: (n_pq × n_pvpq)
J4 = ∂Q/∂V（电压变一点，无功变多少）—— 大小: (n_pq × n_pq)
```

**注意**：PV 节点的电压幅值是固定的，所以 J2 和 J4 只包含 PQ 节点的列。

**2d. 解线性方程组**

```python
dx = np.linalg.solve(J, mismatch)
```

这一步相当于求解：**"电压需要修正多少，才能消除当前的功率失配？"**

```
J * [Δθ, ΔV/V] = [ΔP, ΔQ]
```

**2e. 更新电压**

```python
# 更新相角（PV和PQ节点）
v_ang[pvpq_idx] += dx[:n_pvpq]

# 更新电压幅值（仅PQ节点，用乘法更新更稳定）
v_mag[pq_idx] *= (1 + dx[n_pvpq:])
```

然后回到 2a，继续下一轮迭代。

### 5.3 辅助函数

#### `_calc_power_injection()`——计算功率注入

```python
def _calc_power_injection(v_mag, v_ang, Y):
    for i in range(n):
        for j in range(n):
            G = Y[i,j].real    # 电导（实部）
            B = Y[i,j].imag    # 电纳（虚部）
            theta_ij = v_ang[i] - v_ang[j]  # 相角差

            P[i] += v_mag[i] * v_mag[j] * (G * cos(theta_ij) + B * sin(theta_ij))
            Q[i] += v_mag[i] * v_mag[j] * (G * sin(theta_ij) - B * cos(theta_ij))
    return P, Q
```

就是把潮流方程 "翻译" 成代码。双重循环遍历所有节点对。

#### `_build_jacobian()`——构建雅可比矩阵

对潮流方程求偏导数。公式比较长，但本质是微积分中的求导：

**非对角线元素**（i ≠ j）：
```
∂P_i/∂θ_j = V_i * V_j * (G_ij * sin(θ_ij) - B_ij * cos(θ_ij))
∂P_i/∂V_j = V_i * (G_ij * cos(θ_ij) + B_ij * sin(θ_ij))  （乘以V_j归一化）
∂Q_i/∂θ_j = -V_i * V_j * (G_ij * cos(θ_ij) + B_ij * sin(θ_ij))
∂Q_i/∂V_j = V_i * (G_ij * sin(θ_ij) - B_ij * cos(θ_ij))  （乘以V_j归一化）
```

**对角线元素**（i = j）：
```
∂P_i/∂θ_i = -Q_i - V_i² * B_ii
∂Q_i/∂θ_i = P_i - V_i² * G_ii
... 等等
```

这些公式不需要记，理解"它是在求偏导数"就行。

#### `_calc_branch_flows()`——计算支路功率

收敛后，根据最终电压计算每条线路上流过的功率：

```python
# 把电压转成复数形式
Vi = v_mag[i] * exp(j * v_ang[i])
Vj = v_mag[j] * exp(j * v_ang[j])

# 用π型等值电路计算电流
I_ij = (Vi/t - Vj) * y + Vi * (jb/2) / t²
I_ji = (Vj - Vi/t) * y + Vj * (jb/2)

# 功率 = 电压 × 电流共轭
S_ij = Vi * conj(I_ij)   # 从i到j的功率
S_ji = Vj * conj(I_ji)   # 从j到i的功率

# 损耗 = 两端功率之和（因为方向相反，和就是损耗）
Loss = S_ij + S_ji
```

## 6. 模块三：可视化 (visualization.py)

提供三种图表，帮助直观理解计算结果。

### 6.1 `plot_voltage_profile()` —— 电压分布图

**功能**：柱状图展示每个节点的电压幅值。

**颜色含义**：
- 绿色：0.95 ~ 1.05 pu（正常）
- 黄色：0.90 ~ 1.10 pu（警告）
- 红色：< 0.90 或 > 1.10 pu（越限）

电力系统运行中，电压保持在合理范围内非常重要。

### 6.2 `plot_convergence()` —— 收敛曲线

**功能**：用半对数坐标画出每次迭代的最大失配量。

**看什么**：
- 曲线应该**快速下降**（体现牛顿法的二次收敛）
- 通常 3~5 次迭代就降到极小值
- 如果曲线不下降或震荡，说明算法可能不收敛

### 6.3 `plot_network()` —— 网络拓扑图

**功能**：画出电网的物理连接关系。

**图中的元素**：
- **红色大圆**：平衡节点（Slack）
- **蓝色中圆**：PV 节点（发电机）
- **绿色小圆**：PQ 节点（负荷）
- **线条粗细**：代表功率流的大小
- **标注**：每个节点的电压和相角

## 11. 常见问题

### Q: "Font 'default' does not have a glyph for '\u2212'" 是什么？

这是 matplotlib 的字体警告。因为配置了中文字体 SimHei，但该字体缺少 Unicode 减号字符。**不影响计算结果**，只是图表上的负号可能显示异常。

### Q: UnboundLocalError: cannot access local variable 'BusType'

`import` 语句放在了函数末尾，Python 把它当成局部变量但在使用时还未赋值。解决方法：把 import 移到文件顶部。

### Q: 为什么有些节点不需要求解电压？

- **PV 节点**：电压幅值由发电机控制，是固定的，只需求解相角
- **SLACK 节点**：电压幅值和相角都固定，不需要求解（它是参考点）
- **PQ 节点**：电压幅值和相角都需要求解

### Q: 为什么需要雅可比矩阵？直接迭代不行吗？

"直接迭代"（如高斯-赛德尔法）可以，但收敛很慢，可能需要几十甚至上百次迭代。牛顿法通过雅可比矩阵利用了方程的局部线性信息，实现二次收敛，大幅减少迭代次数。代价是每次迭代需要额外计算雅可比矩阵并求解线性方程组。

### Q: 标幺值 1.0 pu 到底是多少伏？

取决于基准值的选取。比如基准电压选 220kV，则 1.0 pu = 220kV，1.05 pu = 231kV。本项目中所有计算都在标幺值下进行，不需要知道具体的实际值。

### Q: 什么是 "收敛"？

指迭代计算的结果趋于稳定。具体来说，当每个节点的功率失配量（计算值与给定值的差距）都小于设定的容差（如 1e-8 pu）时，认为算法收敛，当前电压就是方程的解。
